## Quiz Game 🎯

A Flutter quiz app that fetches questions from an API and lets users test their knowledge in a simple and interactive way.

## Features
 • Fetch quiz questions from API
 
 • Multiple-choice answers
 
 • Highlight correct and wrong answers
 
 • Score tracking
 
 • Restart quiz

## Tech Stack
 • Flutter
 
 • Dart
 
 • Riverpod (state management)
 
 • HTTP

## What I Learned
 • How to work with APIs

 • Managing state using Riverpod
 
 • Handling UI updates based on state
 
 • Solving layout and logic issues

## Getting Started
 1. Clone the repository
 
 2. Run flutter pub get
 
 3. Run the app
